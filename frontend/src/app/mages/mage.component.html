Mages

Let's build upon the concept of the boot.dev Dragon class. We're going to the creating a series of mages that will have different spells. Just like the dragon return the list of units hit by the spells.

Ice Mage

The ice mage is the most similar to the dragon. It will have a two range values a low, n, and high, m. It will generate two rectangles. One extends vertically in each direction by m and horizontally by n. The other horizontally by m and vertically by m.

Earth Mage

The earth mage will raise a set of spires from it's starting position to a target position (x, y). The first set of spires will be a rectangle of height 1 traveling horizontally until it reaches the target's y. The next rectangle will travel vertically until it reaches the target position with a width of 1.

Fire Mage

The fire mage creates a rectangle that is a height 1 and extends outward by n on both sides, per row. It will continue creating rectangles until that start at y + 1 and extend outward by n - 1 on each side with the last rectangle being of width 2. In the shape of a pyramid.

Water Mage

The water mage will create a wave that starts on a coordinate (x, y) and moves horizontally growing in height by 1 each row until it reaches a height m, and shrinks by 1 each row until it reaches a height of 1. The wave will repeat by a period m.

Dark Mage

The dark mage will cast a haze of darkness that takes the shape of a diamond. The diamond will be the shape of the fire mage's pyramid however, the base will serve as the middle of the diamond, and it will create a pyramid on both sides.

    def __init__(self, name, pos_x, pos_y):
        self.name = name
        self.pos_x = pos_x
        self.pos_y = pos_y

    def in_area(self, x_1, y_1, x_2, y_2):
        return (
            self.pos_x >= x_1
            and self.pos_x <= x_2
            and self.pos_y >= y_1
            and self.pos_y <= y_2
        )
class Dragon(Unit):
    def __init__(self, name, pos_x, pos_y, fire_range):
        super().__init__(name, pos_x, pos_y)
        self.__fire_range = fire_range

    def breathe_fire(self, x, y, units):
        hit_by_blast = []
        for unit in units:
            in_area = unit.in_area(
                x - self.__fire_range,
                y - self.__fire_range,
                x + self.__fire_range,
                y + self.__fire_range,
            )
            if in_area:
                hit_by_blast.append(unit)
        return hit_by_blast
